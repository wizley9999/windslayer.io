# Stat Distribution Guide by Class (Beginner-Oriented)

This guide is written for **new players aiming toward 2nd Job Advancement**.

If you're thinking:

> "I think I invested too much into this stat..."

- If it's **STR (early level)** → You can either restart the character (if very low level) or compensate through later leveling.
- If you're a late-game veteran considering a reset → You probably already know better optimization paths.

**Important:**  
Stat reset is officially provided upon Job Advancement.  
Therefore:

👉 **Before your 1st Job Advancement: Put ALL points into STR.**

---

# Stat Basics (Core RPG Rule)

There are 4 main stats:

- **STR** → Main damage stat for **Warrior / Monk**
- **DEX** → Main damage stat for **Archer / Rouge**
- **INT** → Main damage stat for **Mage / Priest**
- **SPR** → Sub-stat that increases **Defense and HP**

---

# Why SPR Matters

Early game is rough:

- You are often undergeared (Lv.10–13 armor is weak).
- Melee classes get hit constantly.
- AoE grinding (Kiting) causes heavy incoming damage.

Because of that:

👉 **All classes except Archer generally need SPR.**

---

# Archer (DEX Class)

### Early Game

- Go **Full DEX (4 per level)**.

### After 2nd Job → Sniper

If you can start AoE grinding:

- Add SPR until you feel:

  > "Okay, I can survive kitings comfortably."

- After that, invest remaining points into DEX.

Most players follow this pattern:

- Keep SPR around their level or lower.
- Put everything else into DEX.

Examples:

- SPR 33 → DEX 99
- SPR 20 → DEX 112

When AoE grinding becomes inefficient (Lv.80+ areas like high-density maps), some players drop SPR and return to single-target builds — but that's advanced optimization.

---

# Mage / Priest (INT Classes)

### Early Game

- Go **Full INT**.

### Around entering Balderan Mine

You will start needing survivability.

Recommended ratio:

- **INT 3 : SPR 1**
- Keep SPR roughly at or below your level.

Same logic as Archer — enough survivability to grind comfortably.

---

# Melee Classes (Warrior / Rogue / Monk)

These classes take heavy damage.

Recommended ratio:

- **3:1 (Main Stat : SPR)**

Examples:

- Warrior → STR 3 / SPR 1
- Rogue → DEX 3 / SPR 1
- Monk → STR 3 / SPR 1

---

## Gear Tip for Melee

After Lv.16–19:

- Upgrade armor frequently.
- Once you enter Balderan, farming materials should give enough gold for armor upgrades.

Think of buying armor as:

> "Investing gold into SPR through equipment."

Very cost-efficient.

---

# Monk Specific Note

Monk Artist is melee:

- Follow **STR 3 : SPR 1**

Some players wonder:

> "Should I invest in INT because MP feels too low (around 90 MP)?"

In practice:

- Few grinding skills are consistently used.
- Cooldowns are long.
- Unless focusing on PvP, investing in INT is generally unnecessary.

---

# Simple Summary

| Class   | Early Game                    | Mid / Late Direction         |
| ------- | ----------------------------- | ---------------------------- |
| Warrior | Full STR (pre-job)            | STR 3 : SPR 1                |
| Monk    | Full STR (pre-job)            | STR 3 : SPR 1                |
| Rogue   | Full STR (pre-job)            | DEX 3 : SPR 1                |
| Archer  | Full STR (pre-job) → Full DEX | Mostly DEX + some SPR if AoE |
| Mage    | Full STR (pre-job) → Full INT | INT 3 : SPR 1                |
| Priest  | Full STR (pre-job) → Full INT | INT 3 : SPR 1                |
