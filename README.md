# WindSlayer Guide for Global Players

An unofficial resource hub for international players of the Korean version of **WindSlayer**, operated by **Sesisoft**.

This project exists to make the Korean service more accessible to players outside Korea by providing translated official information, gameplay guides, and optional tools that improve usability.

## Project Overview

This repository serves as a centralized information platform designed specifically for global players who:

- Do not speak Korean
- Need translated official announcements
- Want optimized gameplay strategies
- Require setup guidance for smoother access

⠀The project is organized into three primary categories:

### 1. Official News (Translated)

We translate and summarize official announcements published by the WindSlayer team, including:

- Patch notes
- Update details
- Event announcements

All translations are unofficial and provided for convenience only.

### 2. Guides (Tips & Tricks)

This section provides gameplay-focused content such as:

- Leveling routes and optimization
- PvP strategies
- Farming locations
- Drop rate information

Content is written based on gameplay experience, community knowledge, and verified in-game testing when possible.

### 3. Tools & Utilities

To assist global players in accessing and understanding the Korean server, this project may provide:

- VPN setup guidance (informational only)
- Client language patch tool (English only)

All tools are provided for convenience and educational purposes. Use at your own discretion.

Release builds are distributed via GitHub Releases.
